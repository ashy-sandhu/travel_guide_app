# Screenshots Directory

This directory contains screenshots of the Pathio app.

## Required Screenshots

Please add the following screenshots to showcase your app:

1. **splash_screen.png** - App splash screen with Lottie animation
2. **home_screen.png** - Main home screen showing popular and nearby places
3. **search_screen.png** - Search interface with filters
4. **trips_screen.png** - My Trips screen showing trip list
5. **place_details.png** - Place details screen with map and information
6. **account_screen.png** - Account and settings screen

## Screenshot Guidelines

- **Resolution**: 1080x1920 (portrait) or 1920x1080 (landscape)
- **Format**: PNG or JPG
- **Quality**: High resolution, clear and readable
- **Device**: Android phone screenshot preferred
- **Content**: Show real data when possible

## How to Take Screenshots

### Using Android Studio
1. Run the app on an emulator or device
2. Click the camera icon in the toolbar
3. Save the screenshot

### Using Flutter Command
```bash
flutter screenshot --type=android
```

### Using Device
- Android: Power + Volume Down
- Then crop and save to this directory
